# Labyrinth Game Project

Лабиринт сокровищ это текстовая adventure-игра на чистом Python, реализованная в консольном стиле. Вы  отважный исследователь, блуждающий по древнему лабиринту, полному хитрых коридоров, скрытых сокровищ, загадок и ловушек. Собирайте предметы, решайте ребусы, перемещайтесь по комнатам и охотитесь за главным призом  сундуком с сокровищами. Игра использует процедурный подход с функциями, циклами, словарями и структурами данных.

## Запуск
poetry run project
